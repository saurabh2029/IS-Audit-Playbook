# ISO/IEC 27001:2022 — IS Auditor Reference

> **Standard:** ISO/IEC 27001:2022 (Information Security Management Systems)  
> **Publisher:** ISO (International Organization for Standardization) / IEC  
> **Replaces:** ISO/IEC 27001:2013  
> **Certification Body:** Accredited certification bodies (e.g., BSI, Bureau Veritas, DNV)

---

## Overview

ISO/IEC 27001 is the **international standard for Information Security Management Systems (ISMS)**. It provides a framework for establishing, implementing, maintaining, and continually improving information security within an organization. Unlike the RBI framework (mandatory for banks), ISO 27001 is voluntary — but certification demonstrates a mature security posture and is increasingly required by regulators, clients, and vendors.

## Using ISO 27001 as Audit Criteria

When writing IS audit findings, cite specific Annex A controls:

```
Criteria example:
"ISO/IEC 27001:2022, Annex A Control A.8.15 (Logging) requires that 
event logs recording user activities, exceptions, faults, and information 
security events be produced, kept, and regularly reviewed. Logs should 
be protected against tampering and unauthorized access."
```
